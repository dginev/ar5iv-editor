\section*{Runtime Rhai package bindings}

This document loads \texttt{example.sty}, which exists only as
\texttt{example.sty.rhai} --- a package binding written in
\href{https://rhai.rs}{Rhai} and loaded by \texttt{latexml-oxide} at
conversion time. There is no Perl \texttt{.ltxml} file and no recompilation:
the binding is plain script, discovered on the search path.

The binding does two things. It calls \texttt{RequirePackage("xcolor")} so
that \verb|\color| becomes available, and it defines \verb|\foo| to expand to
\verb|{\color{purple}FOO}|.

Here is that macro in action: \foo{}. And once more inside a sentence --- the
word \foo{} should render in purple.
