\section*{Runtime Rhai package bindings}

This document loads \texttt{example.sty}, which exists only as
\texttt{example.sty.rhai} --- a package binding written in
\href{https://rhai.rs}{Rhai} and loaded by \texttt{latexml-oxide} at
conversion time. There is no Perl \texttt{.ltxml} file and no recompilation:
the binding is plain script, discovered on the search path.

It defines \verb|\foo| with a \texttt{DefConstructor} that builds the output
node directly --- an \verb|<ltx:text>| carrying an inline \verb|color:purple|
CSS style. The binding also shows the \texttt{DefMacro} alternative (commented
out): instead of emitting XML, it expands \verb|\foo| to the real TeX source
\verb|{\color{purple}FOO}|, which is why it calls \texttt{RequirePackage("xcolor")}.

Here is \verb|\foo| in action: \foo{}. And once more inside a sentence --- the
word \foo{} should render in purple.
