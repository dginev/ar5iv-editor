Help

Editing & preview

  • Type LaTeX in the source pane; the preview pane re-renders after a 300 ms pause.
  • If your snippet contains \begin{document}…\end{document}, the preamble is split off and sent as preamble.
  • Click the status bar to toggle the conversion log.
  • While a conversion is in flight, the preview pane shows a diagonal-stripe watermark, the cursor turns into the system progress spinner, and the status counter ticks (converting… (Xs)). Big arXiv sources can take 5–15 seconds — this is normal, not a freeze.

The file panel

The leftmost column lists every file in your current session. Click any file to open it in the editor. Switching the active file does not trigger a re-conversion: the engine always renders the project's main entry (auto-detected from 00README.json if present, otherwise via the same heuristic arXiv uses), regardless of which buffer you're viewing.

Each row reveals a small × on hover that deletes the file (or recursively removes the folder, with confirmation). Conversions fire only on events that actually change what gets rendered: edits, uploads, deletes, file creation, clearing, and example swaps.

The four header buttons

create
Prompts for a relative path (e.g. chapter1.tex) and writes an empty file at that location, then opens it in the editor. Useful for sectioning out a longer document.
upload

Click to open a file picker that accepts any number of loose files plus .zip / .tar.gz / .tgz archives — the server unpacks the archive into the session and lays out its files at the project root.

For full directory uploads, drag and drop a folder anywhere onto the file panel (the entire sidebar is a drop target; it lights up while a drag hovers over it). Sub-directory layout is preserved. Drag-and-drop also accepts the same archive formats as the click path.

download
Bundles every file in your session, plus the latest rendered index.html and the ar5iv stylesheet pack, into a single ZIP. Extract it anywhere and open index.html in a browser to see the rendered document offline.
clear
Deletes every file in the project (with a confirmation dialog). The session itself stays open; you can immediately upload a new folder or start fresh with create. The button hovers red to flag the destructive action.

Themes

Click the chrome-theme button at the top-right to cycle through three palettes. Your choice is remembered across reloads.

paper
Light, cream-toned surfaces with a deep-purple accent and serif preview type — the default for screens whose system colour preference is light. Optimised for long-form reading.
midnight
Soft dark surfaces with a lavender accent. The default when your system prefers a dark colour scheme. Easy on the eyes for late-night sessions without going jet-black.
terminal
Near-black surfaces with a phosphor-green accent and a fully monospaced typeface throughout (chrome included). Reads like a retro CRT — useful when you'd rather see structure than typography.
