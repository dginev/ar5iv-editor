ar5iv-editor

A web-based LaTeX editor with a live preview. Source TeX is sent to the backend over a WebSocket; the server runs latexml-oxide (a pure-Rust LaTeXML port) to produce HTML5 with native MathML, and the rendered fragment is morphed back into the preview pane.

This is the Rust successor to the Perl / Mojolicious ltxmojo plugin.
