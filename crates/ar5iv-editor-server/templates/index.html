ar5iv-editor

A Rust port of ltxmojo: a small web frontend for latexml-oxide that converts LaTeX to HTML5 with native MathML.

  • Web Editor

    Live LaTeX-to-HTML preview, served over a WebSocket. Type in the left pane, watch the right pane re-render.

  • Upload & Convert coming soon

    Drop a .tex file (or archive) and download the converted HTML. Not yet wired up in the Rust port.

  • About

    What this project is, where it came from, and how it relates to the original Perl/Mojolicious plugin.

  • Help

    Quick notes on using the editor: preamble splitting, the conversion log, keyboard tips.
